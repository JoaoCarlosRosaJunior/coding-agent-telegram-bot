# Coding Agent Telegram Bot

## How to use?

1. Download the repository
2. Get your telegram and OpenAI API Keys
3. Set those api keys on a file called .env on the lsu_pilot directory

## Running the application

1. Run:
   ``` Bash
   pip install python-telegram-bot openai python-dotenv
   ```

   ``` Bash
   poetry shell
   ```

   ``` Bash
   python lsu_pilot/main.py
   ```
